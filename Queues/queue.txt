A queue is a Interface.

A Queues is a linear data structure in Java that follows the First-In-First-Out (FIFO) principle,
meaning the first element added to the queue will be the first one to be removed.
It has operations such as enqueue, dequeue, and peek. It can be implemented using an Array,
Linked List, or endeavour.

Example :- In a line who get the item first,get removed from the line first.
like in a queue In Macdonald, The very first person who get the burger, get remove form the line very first.
which means first in first out.
